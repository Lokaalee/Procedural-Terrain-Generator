Procedural Terrain Generator
Our group selected a Procedural Terrain Generator as our Procedural Content Generation project because it provides an effective way to apply mathematical concepts such as vectors, geometry and algorithms to solve a real-world problem. Rather than creating landscapes manually, procedural terrain generation uses mathematical equations to generate hills, valleys and other landforms automatically.

We chose this project because it demonstrates how complex environments can be created efficiently using computation while allowing us to explore the mathematical and programming principles behind procedural content generation. It also offers opportunities to develop a realistic and scalable terrain generation system that can be extended with additional features as the project progresses.
